✍️
ZK Writer (Raw TCP)
Write attendance records to device
  • Custom ZKTeco protocol implementation (no pyzk needed)
  • CMD_WRITE_ATTLOG (cmd 79) packet builder
  • ZK time encoding (packed 32-bit integer)
  • Best-effort write with fallback to SQLite
  • Tested: F18, F22, K40, iClock 360/580, UA760, MA300
POST /api/punch/admin/direct

Login System
Admin + employee authentication

  • must_change_password flag on first login

Launcher & Deployment

Windows Launcher (attendance.bat)

  • Auto-detects local IP for dashboard URL
  • Full sync, device-only sync, employee export
  • Today / history absent report generation
  • MDB scan diagnostics
  • Database backup

Linux Launcher (attendance.sh)

  • Remove systemd service
  • Browser auto-open via xdg-open

Install Script (install.sh)

  • Auto MDB employee export if .mdb found
  • Device sync on completion
